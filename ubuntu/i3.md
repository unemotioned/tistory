## Install Packages

- `i3`: the window manager
  - `i3lock`: lock screen
  - `i3status`: statusbar
  - `dmenu`: app launcher
- `alacritty`: i3 sensible terminal
- `feh`: wallpaper
- `picom`: compsiter(fixes screen tearing)
- `playerctl`: media control

```sh
sudo apt install -y i3 alacritty feh picom playerctl
```

## Start i3wm

Logout and select `i3` after selecting the `user`

To logout from i3 session

```sh
i3-msg exit
```

## Mouse sensitivity

To change mouse sensitivity create config file

```sh
sudo -E nvim /etc/X11/xorg.conf.d/40-mouse.conf
```

Just put this in there and reboot

Reference: 10cm from edge to edge on QHD monitor

```conf
Section "InputClass"
    Identifier "Disable accel and slow mouse"
    MatchIsPointer "on"
    Driver "libinput"
    Option "AccelProfile" "flat"
    Option "AccelSpeed" "-0.13"
EndSection
```

## Key Repeat

- delay: 250ms
- repeat speed: 70 frequency

```conf
exec_always --no-startup-id xset r rate 250 70
```

## Dmenu

In i3 config file instead of `dmenu_run` `i3-dmenu-desktop` only shows desktop app

```conf
bindsym Mod1+space exec --no-startup-id i3-dmenu-desktop
```

## Status Bar

Copy default config to `~/.config`

```sh
mkdir -p ~/.config/i3status
cp /etc/i3status.conf ~/.config/i3status/config
```

## Wallpaper

Launch `feh` on login

```conf
exec_always --no-startup-id feh --bg-scale {absolute path to the image}
```

## Monitor Setting

Show information with `xrandr` and `--query` flag

```sh
xrandr -q
```

monitor, res, fps

```sh
xrandr --output DP-4 --mode 2560x1440 --rate 239.97
```

Dual monitor use `--left-of`

```sh
xrandr --output eDP-1 --mode 2560x1440 --rate 239.97 --output HDMI-1 --mode 1920x1080 --rate 60.00 --left-of eDP-1
```

Add the command above with `exec_always --no-startup-id` at `i3/config`

Assign workspace `7` or `y` to secondary monitor
