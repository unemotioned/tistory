# IDE Shortcuts

## JetBrains

### JetBrains - macOS

| Shortcut        | Action                           |
| --------------- | -------------------------------- |
| Cmd + Shift + A | Search Everywhere                |
| Cmd + A         | Select All                       |
| Cmd + 6         | Open TODO tool window            |
| Cmd + 4         | Open Run tool window             |
| Cmd + N         | New file/folder                  |
| Opt + Cmd + L   | Reformat code                    |
| Ctrl + PgUp     | Navigate to previous tab         |
| Ctrl + PgDn     | Navigate to next tab             |
| Opt + F7        | Find Usages                      |
| Shift + F6      | Rename symbol                    |
| Shift + Shift   | Search Everywhere (double Shift) |
| Ctrl + Opt + R  | Run Anything                     |
| Shift + F2      | Go to next problem/error         |

### JetBrains - Windows

| Shortcut       | Action                   |
| -------------- | ------------------------ |
| Ctrl + Alt + S | Open Settings            |
| Alt + 1        | Open Project tool window |
| Alt + 6        | Open TODO tool window    |
| Alt + 4        | Open Run tool window     |
| Alt + Insert   | Generate code            |
| Ctrl + Alt + L | Reformat code            |
| Ctrl + PgUp    | Navigate to previous tab |
| Ctrl + PgDn    | Navigate to next tab     |
| Alt + F7       | Find Usages              |
| Shift + F6     | Rename symbol            |
| Shift + Shift  | Search Everywhere        |
| Shift + F10    | Run                      |
